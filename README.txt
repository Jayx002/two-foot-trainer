How to use:
1) Unzip this folder.
2) Open `index.html` in any modern browser (Chrome, Edge, Safari, Firefox).
3) Use the left sidebar to filter drills, set the timer, and build a session plan.
4) Click **Print** to generate a printable session sheet (you can save as PDF).

Notes:
- Your dominant foot selection, notes, and plan are saved locally in your browser (localStorage).
- The timer has audio beeps generated in your browser (no internet needed).
- This is a demo; you can add more drills by editing the `index.html` (search for `DRILLS`).

Enjoy!
